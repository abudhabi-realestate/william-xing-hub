# 黄金签证-材料与流程

> 适用于**房产投资类**黄金签证申请。  
> 材料清单与审批时效因酋长国、申请渠道略有差异，以 GDRFA / ICA 及当时政策为准；不构成法律或投资建议。

---

## 一、主申请人材料（房地产投资者）

| 序号 | 材料 | 说明 |
|------|------|------|
| 1 | **护照副本** | 有效期满足居留要求 |
| 2 | **房产证（Title Deed）** | 房产价值不低于 **200 万 AED** |
| 3 | **白底免冠照片** | 按签证中心规格 |
| 4 | **付款证明** | 视付款方式二选一：<br>· **期房：** Account Statement（账户流水 / 付款记录）<br>· **贷款：** 银行出具的 **NOC**（No Objection Certificate，无异议证明） |
| 5 | **本地医疗保险** | 阿联酋认可的医疗保险保单 |
| 6 | **家庭成员护照副本** | 如有家属一并申请，见下文 |

---

## 二、家庭成员材料

家属护照副本之外，按关系补充以下文件（**均需公证认证**）：

| 关系 | 所需材料 |
|------|----------|
| **夫妻** | 结婚证明 |
| **子女** | 出生证明；**18 岁以上**子女另需**未婚证明** |
| **母亲为主申请人（Sponsor）** | 父亲出具**不反对函** |
| **父母** | Sponsor 的**出生证明**（用以证明亲子关系） |

> **公证认证：** 上述境外文件通常需经本国公证 + 阿联酋驻当地使领馆认证，具体以申请窗口要求为准。

---

## 三、整体流程

| 步骤 | 内容 | 参考时效 |
|------|------|----------|
| 1 | **提交文件** | 通过开发商通道、注册经纪或 GDRFA / ICA 指定窗口递交 |
| 2 | **政府审批** | 约 **半天至 1 天** |
| 3 | **体检** | 约 **半天**（指定医疗中心） |
| 4 | **指纹录入** | 约 **半天** |
| 5 | **领取签证 / Emirates ID 物理卡** | 常规数个工作日；**付费加急可当天** |

---

## 四、政府规费（迪拜 · 10 年黄金签证）

> 以下为 **Dubai Land Department（DLD）** 及居留审批相关**政府规费**，实际金额因是否加急等因素可能略有浮动；**阿布扎比**收费结构可能不同，以递交窗口为准。不含医疗保险、文件公证认证、中介服务费等。

### 主申请人 · 投资签证（10 年）

| 费用项 | 金额（AED） |
|--------|------------|
| DLD Fees Visa | 4,020.00 |
| Visa Processing | 2,709.90 |
| Medical VIP | 705.86 |
| Emirates ID | 1,167.12 |
| Admin Fees | 1,155.00 |
| **合计** | **9,757.80** |

附注：若当前签证下已有家属（Dependent），每位另收 **100 AED**。

### 家属 · 家庭签证（10 年）

| 费用项 | 金额（AED） |
|--------|------------|
| Visa Processing | 2,709.90 |
| Medical VIP | 705.86 |
| Emirates ID | 1,167.12 |
| Admin Fees | 840.00 |
| **合计** | **5,422.88** |
| Open File（一次性） | 318.75 |

附注：申请人 **未满 18 岁**，合计 **4,717.45 AED**（不含 Open File）。

---

## 五、与申请门槛的对应关系

材料准备前，请先确认付款节点是否满足所在酋长国要求（详见 [黄金签证-常见问题](./golden-visa-guide.html)）：

- **迪拜：** 付清首付后可申请（按揭需银行 NOC）
- **阿布扎比：** 累计支付至少 **200 万 AED** 后可申请
- **多套房产：** 可合计达到门槛
