# 迪拜 · CRS 与税务合规 — 常见问题

> 基于 OECD CRS / CRS 2.0、CARF、IPI MCAA 公开框架，以及阿联酋联邦税务局（FTA）相关规定整理。  
> 仅供参考，不构成税务、法律或投资建议。具体情况请咨询持牌专业顾问。

---

## 一、先搞懂：CRS 是什么？

**Q1：CRS 和我有什么关系？**

只要你在中国仍是税务居民，又在境外（包括迪拜）持有金融账户——银行存款、证券、基金、保险（有现金价值）等——相关账户信息就可能通过 CRS 交换给中国税务机关。

CRS 交换的是**账户信息**（余额、年度利息/股息/其他收入、资产出售所得等），不是逐笔交易流水。收到信息后，税务机关会比对你是否已就境外所得依法申报。

**Q2：CRS 和 FATCA 是一回事吗？**

不是。FATCA 是美国的单向申报体系，管美国税务居民。CRS 是 OECD 推动的多边交换机制，已有 120 多个国家和地区参与。美国未加入 CRS；阿联酋银行对美国账户持有人走 FATCA。

**Q3：CRS 会让我「本金被征税」吗？**

不会。常见误解。CRS 报告账户余额和年度收入汇总；是否交税、交多少，取决于各国税法对你**所得**的认定。余额本身不是税基，但未申报的利息、股息、转让所得等，可能触发补税。

**Q4：100 万人民币以下就不会被交换吗？**

这是误解。100 万人民币（或等值）门槛仅影响部分存量账户的**加强尽职调查程序**，不豁免信息交换。只要被识别为某国税务居民，账户原则上都应申报，与金额大小无关。

## 二、CRS 2.0：2026 年起要注意什么？

**Q5：CRS 2.0 和 1.0 有什么本质区别？**

四个方向同时收紧：

1. **范围更宽** — 电子货币账户（如 PayPal、Wise 等，余额超一定门槛）、央行数字货币、通过机构间接持有的加密资产等，纳入报告。
2. **多重税务居民全报** — 废除「二选一」：有几个税务居民身份，就向几个国家交换；银行还会追问是否在其他国家每年停留超过 90 天、是否在其他地方报过税。
3. **信托穿透更细** — 须申报控制人具体角色；代付学费、免息贷款、核销欠款等「间接分配」明确须申报。
4. **投资入籍/居留加强审查** — 黄金签证、护照项目等可能触发银行增强尽职调查（EDD）。

**Q6：迪拜什么时候实施 CRS 2.0？**

根据 OECD 公开时间表，阿联酋计划在 **2028 年** 切换至 CRS 2.0；CARF（加密资产报告）也预计在相近窗口启动交换。欧盟、英国等已从 2026 年 1 月起实施，趋势会逐步覆盖主要金融中心。

**Q7：加密资产能通过搬到迪拜来「躲开」CRS 吗？**

不能指望。CARF 专门覆盖加密货币、稳定币、可交易 NFT 等，与 CRS 2.0 互补。在阿联酋，合规路径是如实申报、保留记录；在中国境内，加密货币交易本身仍受当地法规约束。把资产换成比特币不是合规方案。

**Q8：迪拜的房产会被 CRS 直接上报吗？**

**房产本身不在 CRS 申报范围内。** 但以下情况仍与合规相关：

- 租金、售房回款进入银行账户 → 账户信息可能被 CRS 交换；
- 中国税务居民：境外出租、转让所得仍须按中国税法申报；
- OECD 另推 **IPI MCAA**（海外不动产信息交换），首批交换约在 2029–2030 年，中国尚未加入首批，但长期趋势需关注。

**误区提醒：**「CRS 2.0 从 2026 年起直接报房产」——不对。房产信息交换是 IPI MCAA，与 CRS 是不同框架。

## 三、迪拜税务居民（TRC）

**Q9：什么叫阿联酋税务居民证书（TRC）？**

TRC（Tax Residency Certificate）由阿联酋联邦税务局签发，用于证明个人或企业在特定年度内满足阿联酋税务居民条件。银行做 CRS 尽职调查、部分国家申请税收协定待遇时，会要求提供这类证明。

**注意：** TRC 是**证明文件**，不是「免税牌照」，也不能自动取消你在其他国家的税务居民义务。

**Q10：在迪拜住多久才能申请 TRC？**

根据 FTA 规定，按**实际在阿联酋居住天数**分三档：

| 情况 | 居住天数 | 主要材料 |
|------|----------|----------|
| 一 | 超过 183 天 | 护照 + 出入境记录 |
| 二 | 90–183 天 | 上述材料 + 阿联酋收入来源 + 固定居所（房产证 / EJARI 租约 / 水电账单等） |
| 三 | 不足 90 天 | 证明财务利益中心与生活重心在阿联酋 + 固定居所 |

**Q11：拿了迪拜黄金签证，是不是自动就是阿联酋税务居民？**

**不是。** 黄金签证解决的是「能否长期合法居留」，不能证明「你实际住在阿联酋」。

- 申请 TRC：仍按实际居住天数准备材料，黄金签证**没有任何加成**；
- 银行开户：持 5 年及以上长期签证（含黄金签证），往往触发**增强尽职调查**——银行会追问是否通过投资居留取得身份、是否同时是其他国家税务居民，并要求收入证明、住址证明等。

**Q12：怎样才能「算」迪拜税务居民？**

实务上需同时满足**法律层面**和**事实层面**：

- **法律：** 达到 FTA 规定的居住天数或重心测试，取得 TRC；
- **事实：** 真实在阿联酋生活、经营或就业，有固定居所、本地银行、生活痕迹；
- **合规：** 如仍是中国税务居民，须在中国就全球所得依法申报；如已成功变更税务居民身份，须确保真正切断中国的习惯性住所联系，而非「人还在国内、身份报迪拜」。

仅买房、偶尔来访、或只把资产转过去，**不足以**构成合规的税务居民安排。

**Q13：TRC 能让我境内的境外账户不再报回中国吗？**

**不能自动做到。** CRS 按你在开户时申报的、以及银行核实后的**税务居民身份**交换信息。

- 若你仍是中国税务居民 → 境外账户（含迪拜账户）仍可能报回中国；
- 若你已成为阿联酋税务居民且银行接受该申报 → 账户信息交换方向会改变，但你需要在**中国端**依法完成税务居民身份变更、清税等手续，不能只做「迪拜开户填阿联酋」。

虚假申报或资料不实，在阿联酋和中国都可能构成严重法律风险。

## 四、在迪拜开户与银行合规

**Q14：阿联酋银行会不会把账户信息报回中国？**

**会，如果你被识别为中国税务居民。** 阿联酋 2018 年起实施 CRS，采用**最宽口径（Widest Approach）**——银行须识别非阿联酋税务居民（美国人除外，走 FATCA），并向其税务居民所在国交换信息。

中国与阿联酋均在 CRS 交换网络内。

**Q15：开户时银行会看什么？**

常见核查线索包括：

- CRS 自我声明表填写的税务居民国；
- 身份证件签发国、现居地址、电话号码区号；
- 有效阿联酋居留签证（若声称阿联酋税务居民）；
- 转账路径、关联账户、历史记录等。

存在多国线索时，银行可要求补充证明；信息不实可能拒开、限制或上报。

**Q16：账户里有多少钱，银行才会「盯更严」？**

阿联酋 CRS 指引对个人账户有分级：

- **不超过 100 万美元：** 基础电子记录排查；
- **超过 100 万美元：** 全面增强审查，含纸质档案；客户经理若知悉你在其他国家有税务居民身份，有主动上报义务。

各银行内部合规可能设置更低门槛，实务中不要假设「小账户没人管」——CRS 本身没有豁免金额。

**Q17：用中国护照在迪拜开户，会被当成中国税务居民吗？**

护照是重要识别指标之一，但不是唯一依据。银行综合地址、签证、自我声明、生活痕迹等判断。中国护照持有人若常住迪拜、持有 TRC、生活重心在阿联酋，可依法申报为阿联酋税务居民——前提是**事实与材料一致**，而非仅因「在迪拜开了户」。

## 五、迪拜房产与资产安排

**Q18：在迪拜买房，和 CRS / 税务合规有什么关系？**

分几层看：

**房产本身：** 不在 CRS 金融账户申报范围内；产权证在你名下，不依赖券商或 App 通道。

**与 CRS 相关的部分：** 租金收入存入迪拜银行账户、售房回款进账 → 相关账户可能被交换；若仍是中国税务居民，须在中国申报境外所得。

**本地税务：** 阿联酋无个人所得税、无资本利得税（个人层面），但仍有物业相关费用、注册费、社区费等，以及来源国（如中国）的申报义务。

**Q19：迪拜房产适合作为「抗通道风险」的配置吗？**

从合规视角，房产与证券的逻辑不同：前者是物理持有、决策自主；后者是账户权利，受通道与监管直接影响。2026 年境内非法跨境证券渠道收紧，与 CRS 信息交换是同一方向的互补动作——**不是说法规要没收资产，而是纸面金融资产的合规成本在上升。**

这不等于「买房能避税」；买迪拜房仍需考虑资金来源合法、购汇合规、持有期税务与长期政策风险。

**Q20：迪拜和阿布扎比，税务上有区别吗？**

**联邦层面一致：** 均无个人所得税、均无资本利得税（个人），CRS 与 TRC 规则适用于整个阿联酋。

**差异在资产与市场：** 供应节奏、价格周期、产业逻辑不同——属于配置选择，不是税务身份选择。税务规划应在联邦框架下统一考虑，而非假设两 emirate 适用不同税法。

**Q21：通过迪拜公司持有房产，能规避 CRS 吗？**

**不能。** 公司账户属于实体账户；若公司被认定为「消极非金融机构」（Passive NFE），须穿透申报实际控制人信息。BVI、开曼等架构在 CRS 下同样适用穿透规则。代持、多层嵌套不会消除申报义务，反而可能触发更严审查和反洗钱关注。

## 六、与中国税务居民的交叉问题

**Q22：家人在国内，本人长期待在迪拜，还算中国税务居民吗？**

**很可能算。** 中国判定税务居民，不仅看 183 天——有住所（户籍、家庭、主要经济利益在中国）即可构成居民。配偶子女在国内，是判断「住所」的重要因素；即使你在迪拜不满 183 天，仍可能被视为中国税务居民，境外账户信息报回中国。

**Q23：「在中国不住满 183 天，境外收入就与中国无关」——对吗？**

**对多数持中国户籍、家庭利益在国内的人，这是危险误解。** 183 天规则主要适用于**无住所**的个人。有住所的中国公民，无论在中国住多少天，通常仍是中国税务居民，须就全球所得在中国申报。

**Q24：境外收入已经在外国交过税，回国还要交吗？**

若仍是中国税务居民，全球所得在中国有申报义务；境外已缴税款可在限额内申请**税收抵免**，不是「境外交过就与国内无关」。需保留完税凭证，按中国税法换算申报。

**Q25：收到税务通知或发现以前漏报，该怎么办？**

建议尽快：

1. 核实通知真实性（谨防诈骗，可到主管机关当面确认）；
2. 整理境外账户对账单、收入证明、完税凭证；
3. 咨询同时熟悉中国与阿联酋税法的专业人士；
4. 在合规框架内主动补报，通常优于被动稽查后的处理。

2025 年以来，中国税务机关已加大 CRS 数据运用；2026 年指令侧重 2022–2024 年度收入核查。窗口期仍在，但不宜观望。

## 七、合规底线：什么绝对不能做？

**Q26：有哪些「伪合规」做法风险极高？**

以下在实务中反复出现，且多为明确红线：

| 做法 | 风险 |
|------|------|
| 亲友代持账户 | CRS 穿透；可能涉及虚假申报、反洗钱 |
| 年底清空账户、年初转回 | 交易监控系统（TMS）红旗信号 |
| CRS 自我声明填假信息 | 多国可能构成刑事犯罪 |
| 分拆购汇（「蚂蚁搬家」） | 违反外汇管理规定 |
| 仅办签证/买房、不实际居住却声称迪拜税务居民 | 银行 EDD、GAAR（反避税）风险 |
| 用小国护照开户只报新身份 | 银行综合核查，不能改变真实税务居民判定 |

**Q27：合规路径大概是什么样的？**

四步框架（适用于在迪拜生活或配置资产的人）：

1. **确认税务居民身份** — 在中国、阿联酋（及其他相关国家）分别是什么身份？是否双重居民？
2. **盘点全球账户与资产** — 银行、证券、保险、信托、房产现金流；
3. **审查申报记录** — 各辖区是否已如实申报？CRS 表格是否准确？材料是否齐全？
4. **制定行动方案** — 该补报则补报，该变更身份则走合法程序；必要时同时咨询中阿两侧顾问。

合规成本通常可控；被查后的补税、滞纳金（中国：日万分之五起）和罚款（偷税可处 0.5–5 倍），才是不确定性来源。

## 八、迪拜视角：怎么理解这些规则？

**Q28：迪拜还值得作为长期布局方向吗？**

从合规框架看，迪拜/阿联酋的优势在于：**零个人税、零资本利得税（个人）、无外汇管制、生活与经商环境开放**，且在 CRS 下若真实成为阿联酋税务居民，账户信息交换方向会相应调整。

但优势不等于「监管真空」：CRS、AML、银行 EDD、来源国申报义务都在。TRC 的价值是**把身份说清楚**，不是「藏资产」。

地缘与政策风险始终存在；合规的长期主义，是真实居住 + 合法身份 + 如实申报，而不是短期「身份包装」。

**Q29：我应该先办 TRC，还是先理清中国端身份？**

取决于你的事实情况，但顺序上建议：

- 先确认**实际生活重心**在哪里、是否具备变更税务居民的事实基础；
- 再决定是否在阿联酋申请 TRC、如何向银行申报；
- 若涉及中国身份变更或清税，在**变更前**完成专业评估，避免「身份切了、义务没清」。

**Q30：什么时候该找专业人士？**

以下情况建议咨询（中阿双边经验优先）：

- 在多国均有账户或房产；
- 保留中国户籍但长期在迪拜生活；
- 持有黄金签证或考虑 TRC；
- 曾用境外账户但未向中国申报；
- 计划出售迪拜房产或大额资金跨境流动；
- 收到税务机关或银行合规问询。
